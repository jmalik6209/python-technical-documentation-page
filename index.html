Introduction

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics. Its high-level built in data structures, combined with dynamic typing and dynamic binding, make it very attractive for Rapid Application Development, as well as for use as a scripting or glue language to connect existing components together. Python's simple, easy to learn syntax emphasizes readability and therefore reduces the cost of program maintenance. Python supports modules and packages, which encourages program modularity and code reuse. The Python interpreter and the extensive standard library are available in source or binary form without charge for all major platforms, and can be freely distributed.

Often, programmers fall in love with Python because of the increased productivity it provides. Since there is no compilation step, the edit-test-debug cycle is incredibly fast. Debugging Python programs is easy: a bug or bad input will never cause a segmentation fault. Instead, when the interpreter discovers an error, it raises an exception. When the program doesn't catch the exception, the interpreter prints a stack trace. A source level debugger allows inspection of local and global variables, evaluation of arbitrary expressions, setting breakpoints, stepping through the code a line at a time, and so on. The debugger is written in Python itself, testifying to Python's introspective power. On the other hand, often the quickest way to debug a program is to add a few print statements to the source: the fast edit-test-debug cycle makes this simple approach very effective.

Hello, World!

Once Python is installed, open up a code editor or Python's built-in IDLE (Integrated Development Environment) and type: >>> print("Hello World") Output: Hello World

Congratulations, you have written your first program! Python's print command takes whatever is inside the quotations and displays it on screen. You can print regular text (called a string, or str), numbers (called integers, or int), numbers with decimals (called floating point numbers, or float), and the values of variables or functions.

Examples: >>> print("Some string") #Use hashtags to write comments in Python: code that computer ignores but instead are like notes which humans read when taking a look at your code for understanding. >>> print(1) #returns an "int", or integer (number without decimal) >>> print(6.9) #returns a "float", or floating point number (essentially a decimal number)

Try writing some more yourself, and experiment with it! Challenge: Can you write a few sentences about yourself and have Python print it out in the terminal?

Variables

Variables in Python are used to store data for later use, and are very commonly used when writing programs. In Python, there is no command for declaring a variable. Rather, it's created the moment you assign a value to it. To assign a value, follow this format:>>> x = 5 #variable with integer value >>> name = "Jeff" #variable with string value >>> print(x) >>> print(name) Output: 5 JeffHere you can see that the stored values of the variables are printed out in the terminal. There are many other use cases which are more specific as well. It is considered best practice to give descriptive variable names, and variable names must start with a letter or underscore character. A variable name can't start with a number, can only contain alpha-numeric characters and underscores, and names are case-sensitive.

The value of a variable can be changed, hence making it variable, and subject to change. Variables can also be assigned as values to other variables. For example: >>> x = 5 # value of the integer 5 is assigned to the variable x. >>> x = 6 # the variable's value is changed and reassigned to the integer 6. >>> y = x # a new variable y is created and is equivalent to the value of variable x >>> print(y) # the value of y is printed out in the terminal Output: 6 These are all the basics of variables. Try and experiment with them yourself to see how they can be used and manipulated.

Data Types

In programming, data types are an important concept as they properly help differentiate what the content of a value is, whether it is a number or text. Python has a few basic data types by default:

  1. Text type, or str: this is the data type which is used to describe strings, or text content.
  2. Numeric types, or int, float, complex: These are data types with the content of numbers. int describes integers, or numbers without decimals, float describes floating point numbers, or numbers with decimals, and complex describes complex numbers, which may involve arithmetic, as well as other more advanced mathematical sequences.
  3. Sequence types, or list, tuple, range: These are used to store multiple data types/values (typically related) together in a sequence for storage.
  4. Mapping type, or dict: This data type is like a database with no memory, as it maps a key to a value like a dictionary in real life would map a word to a definition.
  5. Set types, or set, frozenset: These data types are similar to sequence types, in that it is an unordered collection type which is iterable, mutable, and has no duplicate elements.
  6. Boolean type, or bool: This data type has only two possible values: True or False. It is typically used to check whether a certain rule or portion of code is true or false.
  7. Binary types, or bytes, bytearray, memoryview: These more advanced types are typically used when dealing with binary numbers or if the program you are creating requires the involvement of binary, hexadecimal, or hashed data. For most general use cases, these aren't used that often.
  8. None type, or NoneType: Similar to the function of null in JavaScript, the NoneType data type just means that there is no real value or data.

If/Else Statements

Python uses conditional If/Else statements for controlling the flow of the program. These conditions are applied so the program can decide what to do next, especially if there are multiple possible outcomes. The usual logical conditions are take from mathematics and used in Python to set up these if/else statements:

  • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than or equal to: a >= b
These conditions are used in several ways within loops to describe situations where a certain outcome is given as the output. Below are some examples: a = 100 b = 75 if a == b: #condition saying if variables a and b are equal, the program will return text print("Equal!") elif (a > b) and (a != b): #elif stands for else-if, adds another condition saying if a is greater AND not equal to b, the program will return text print("A is greater/equal to B!") elif (b > a) or (b == a) #elif condition saying if b is greater OR equal to a, the program will return text print("B is greater than/equal to A!") else: #condition that if no condition matches, the program returns text print("No matches!") This is a basic example of conditions and If/Else statements in Python. The output isn't included, so, without copying the code and running it, can you figure out what the program will return?

Loops

There are two types of loops in Python: For Loops and While Loops.

For Loops

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string). It can be used when you are looping over a set number of times that you know, and ends once that number of loops is done. If you wanted to loop over lists, for example, you could do this: >>> fruits = ["apple", "banana", "cherry"] >>> for x in fruits: >>> print(x) Output: apple banana cherry Or, if you wanted to loop through letters in a word, you could do: >>> for x in "fruit": >>> print(x) Output: f r u i t To loop through a set of code a specified number of times, we can use the range() function. The range() function returns a sequence of numbers, starting from 0 by default, and increments by 1 (by default), and ends at a specified number. Like this:Increment the sequence with 3 (default is 1), start sequence from 2 (default is 0), end at 30: >>> for x in range(2, 30, 3): >>> print(x) What do you think the output for this block of code will be? Try it yourself!

While Loops

With the while loop we can execute a set of statements as long as a condition is true. For example, if you want to print a number, i, as long as it is less than 6, you can write something like this:>>> i = 1 >>> while i < 6: >>> print(i) >>> i += 1 #increments i by 1 so it keeps counting up and the loop won't repeat Output: 1 2 3 4 5 There are some other conditions used in both for and while loops. These are break, continue, and else. Examples:#Break statement stops the loop if a certain condition is met. In this case, exits loop when i = 3 i = 1 while i < 6: print(i) if i == 3: break i += 1 #Continue statement stops current iteration of the loop and continues with the next. In this case, continues when i = 3 i = 0 while i < 6: i += 1 if i == 3: continue print(i) #Else statement after a while loop prints a message once the condition is false. i = 1 while i < 6: print(i) i += 1 else: print("i is no longer less than 6") Try these all yourself and experiment with the outputs! See if you can find different conditions to implement into your for and while loops.

Functions

A function is a block of code that only runs when it is called. They are useful in conditional programming/control flow when you may not need or want part of a program to run throughout. You can pass data, known as parameters, into a function, which returns data as a result. In Python a function is defined using the def keyword, and called by using the function name followed by parentheses: def my_function(): print("Hello from a function") my_function() #calls function to be used Output: Hello from a function Another part of functions is their arguments which are put in their parentheses once they are defined. You can add as many arguments as you want, just separate them with a comma. The following example has a function with one argument (firstname). When the function is called, we pass along a first name, which is used inside the function to print the full name:def my_function(fname): print(fname + " Billings") my_function("Jeff") my_function("Marcus") my_function("Linus") Output: Jeff Billings Marcus Billings Linus Billings These are the basic features of a function. Within them, you can include any code you want like regular programs would have, such as loops, conditional statements, various data types, etc. Functions are one of the biggest parts of programming and will help you go from a Python beginner to someone more experienced as you learn how to utilize them.
